Installation
--------------------------------------------------------------------------------

This code supports installation using pip (via setuptools). To install from the
git repository:

.. code-block:: bash

    git clone https://github.com/asherbender/bayesian-linear-model
    cd bayesian-linear-model
    sudo pip install .

Dependencies
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

* Python
* Numpy
* Scipy
* Sphinx
